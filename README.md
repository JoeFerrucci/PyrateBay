# PyrateBay
Download from ThePirateBay using Python

Eddie told me he wanted a way to automate his Suits downloads from ThePirateBay. This is the answer to his request.

# Requirements
 - Python 3
 - Pip 3
  - lxml
  - requests

# Usage:
`$ python3 pyrate.py suits`
